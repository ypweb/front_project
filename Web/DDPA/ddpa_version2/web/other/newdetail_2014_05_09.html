贷贷平安的贷款产品主要分为抵押贷款跟信用贷款

抵押贷款包括车易贷和房易贷

1、 车易贷（方便快捷，有车就能贷 ，不押车，最快四小时放款）

申请条件：

  • 20-55周岁的中国大陆公民
  • 拥有对该车辆的所有权
  • 业务开展城市长期居住和工作（目前仅接受深圳地区客户）

必要申请材料

  • 身份证（非本地户籍需同时提供居住证或暂住证）
  • 机动车登记证、行驶证
  • 车辆商业保险全险保单（有效期覆盖合同期的车辆保险单）
  • 工作证明等其他资信材料

产品条款说明

  • 借款额度： 1-200万
  • 借款期限 ：1-36个月
  • 借款年利率：9%-24%
  • 还款方式：等额本息、先息后本
  • 信用贷款分为薪易贷、商易贷

信用贷款包括薪易贷和商易贷

薪易贷（有信用就有钱！ 动一动鼠标，让您的财富动起来！）

申请条件：

  • 20-55周岁的中国大陆公民
  • 在现单位工作满6个月
  • 月收入4000以上

必要申请材料

  • 身份证（非本地户籍需同时提供居住证或暂住证）
  • 央行个人征信报告（当地人民银行打印或登录互联网个人信用信息服务平台下载）
  • 带公司公章的劳动合同或工作证明
  • 近6个月工资卡银行流水及社保记录

产品条款说明

  • 借款额度：3,000-500,000 元
  • 借款期限：1-36个月
  • 借款年利率：9%-24%
  • 还款方式：等额本息，每月还款

2、 商易贷（活用资金，让您的生意飞起来！）

申请条件：

  • 20-55周岁的中国大陆公民
  • 店铺或企业经营超一年

必要申请材料

  • 身份证（非本地户籍需同时提供居住证或暂住证）
  • 央行个人征信报告（当地人民银行打印或登录互联网个人信用信息服务平台下载）
  • 经营证明
    通过年检且注册满1年的营业执照
    经营场地租赁合同＋90天内的租金发票+水电单据
  • 银行流水 可体现经营情况的近6个月常用银行卡流水（对公账户或个人账户）

产品条款说明

  • 借款额度：3,000-500,000 元
  • 借款期限：1-36个月
  • 借款年利率：9%-24%
  • 还款方式：等额本息，每月还款
